# 🍽️ Restaurant Menu Card

A responsive restaurant menu web app built using **HTML, CSS, and JavaScript**. Users can browse menu items, adjust item quantities, and place an order with instant confirmation feedback — all without any external libraries or frameworks.

## ✨ Features

- 🍔 Interactive menu cards displaying food items with images, names, and prices
- ➕➖ Quantity controls to increase or decrease item count before ordering
- 🛒 Order submission with real-time quantity tracking
- ✅ On-screen confirmation feedback after placing an order
- 📱 Fully responsive design that works across desktop, tablet, and mobile
- 🎨 Clean, modern UI built with pure CSS (no frameworks)

## 🚀 Getting Started

To run this project locally:

1. Clone the repository
```bash
   git clone https://github.com/komalakonda/Restaurant-Menu-Card.git
```
2. Navigate into the project folder
```bash
   cd Restaurant-Menu-Card
```
3. Open `index.html` in your browser — no build tools or dependencies required.
